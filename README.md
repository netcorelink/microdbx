`netcorelink/microdb` is a Lightweight key-value database for microcontrollers with Flash and RAM storage backends.

## Install by `pip` `microdb`

```bash
pip install microdb
```

## Install by `MicroPython` `microdb`

For MicroPython, installation is performed by copying `microdb/*` the library files to the device's file system.

example: `microdb/*` - `:lib/*`
